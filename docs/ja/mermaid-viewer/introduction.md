# Mermaid Viewerについて

[Mermaid Viewer](https://mermaidviewer.com)は、開発者向けに設計されたオンラインダイアグラム作成ツールで、ユーザーがアイデアや概念を簡単に視覚化できるよう支援します。Mermaid構文を使用して専門的な技術図を作成・共有する最も簡単な方法です。

### 製品概要

[Mermaid Viewer](https://mermaidviewer.com)を使用すると、ソフトウェアをインストールすることなく、ブラウザ上で直接様々な種類の図を作成、編集、共有することができます。このツールはリアルタイムプレビュー機能を提供し、Mermaidコードを記述すると「レンダリング」ボタンをクリックすることなく、図の変更をすぐに確認できます。

### 主な機能

- **リアルタイムプレビュー**：入力時に即座に変更が視覚化され、「レンダリング」ボタンが不要
- **AI支援コーディング**：GPT-4を使用してテキスト記述をMermaidコードに変換
- **スマートエクスポート**：SVG/PNG/Markdown形式で保存、DPIやダーク/ライトテーマをカスタマイズ可能
- **企業向け統合**：Dockerによるプライベートデプロイメント + ロールベースのアクセス制御
- **インテリジェントデバッグ**：構文エラーのハイライト表示と自動修正提案
- **リアルタイムコラボレーション**：チームメンバーとエディタリンクを共有して図を共同編集

### サポートされている図の種類

[Mermaid Viewer](https://mermaidviewer.com)は30種類以上の図をサポートしています：
- フローチャート
- シーケンス図
- クラス図
- 状態遷移図
- ER図（エンティティ関係図）
- ユーザージャーニー
- ガントチャート
- 円グラフ
- マインドマップ
- タイムライン
- その他多数...

### Mermaid Viewerを選ぶ理由

- **リアルタイムプレビュー**：入力時に即座に図の変更を確認できる
- **登録不要**：アカウント作成なしですぐに図の作成を開始できる
- **プライバシー重視**：データはブラウザに保存され、サーバーにアップロードされない
- **使いやすさ**：3つの簡単なステップで専門的な図を作成
- **完全無料**：基本機能は無料で使用可能、隠れた費用なし

[Mermaid Viewer](https://mermaidviewer.com)は、チームコラボレーションや技術文書作成に最適なツールであり、複雑な図の作成プロセスを数ステップに簡略化し、アイデアを素早く視覚化するのに役立ちます。
